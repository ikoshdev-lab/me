# Creata Portfolio Veb-sayti

Bu mening shaxsiy portfolio veb-saytim bo'lib, "Creata" brendi ostida yaratilgan. Loyiha mening ko'nikmalarim, xizmatlarim va kelajakdagi maqsadlarimni namoyish etish uchun mo'ljallangan.

## ✨ Asosiy Xususiyatlari

- **To'liq Moslashuvchan (Responsive):** Barcha qurilmalarda (desktop, planshet, mobil) mukammal ishlaydi.
- **Ko'p tillilik:** O'zbek, Rus va Ingliz tillarida mavjud.
- **Backend bilan integratsiya:** Aloqa formasi Node.js (Express.js) orqali ishlaydi va xabarlarni to'g'ridan-to'g'ri emailga yuboradi.
- **Zamonaviy Animatsiyalar:** AOS, Typed.js, 3D Tilt, Text Scramble kabi effektlar bilan boyitilgan.
- **Interaktiv Elementlar:** AI Chatbot, Custom Cursor, Lightbox va boshqalar.
- **Bahoriy Dizayn:** Navro'z bayramiga moslashtirilgan maxsus dizayn, gullar effekti va fon musiqasi.

## 🚀 Texnologiyalar Steki

### Frontend
- HTML5
- CSS3 (Variables, Grid, Flexbox)
- JavaScript (ES6+)
- AOS (Animate on Scroll)
- Swiper.js (Sliderlar uchun)
- Typed.js (Yozuv effekti)
- Vanilla-Tilt.js (3D effekt)

### Backend
- Node.js
- Express.js
- Nodemailer (Email yuborish uchun)
- Express-rate-limit (Spamdan himoya)

## ⚙️ Lokal Ishga Tushirish

1.  `backend` papkasiga kiring: `cd backend`
2.  Kerakli kutubxonalarni o'rnating: `npm install`
3.  `.env` faylini o'zingizning ma'lumotlaringiz bilan to'ldiring.
4.  Serverni ishga tushiring: `npm run dev`
5.  Asosiy papkadagi `index.html` faylini brauzerda oching.
